#  音声・インパルス応答データ

## 音声データ

- cleaner.wav

   掃除機騒音の音声wavデータ．  
   フィードフォワード制御のシミュレーションで使用．  
   - サンプリングレート：16k Hz  
   - チャンネル：モノラル
   - 量子化ビット幅：16bit
   
- harmonics.wav

   機械騒音＋狭帯域騒音の音声wavデータ．  
   フィードバック制御のシミュレーションで使用．
   - サンプリングレート：16k Hz  
   - チャンネル：モノラル
   - 量子化ビット幅：16bit
